# Hugo Notecard Project

Readable card-style Hugo theme by Gladushenko for personal blogs, notes, and technical writing.

Repository: <https://github.com/gladushenko/hugo-theme-notecard>

Demo: 

## Features

- Responsive card-based layouts
- Light and dark color schemes
- Search
- Tags and categories
- Table of contents
- Image galleries
- Code highlighting

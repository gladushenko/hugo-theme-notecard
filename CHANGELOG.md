# Changelog

All notable changes to Hugo Notecard Project will be documented in this file.

## 1.0.0 - 2026-07-14

Initial public release.

### Added

- Card-based layouts for home, list, taxonomy, article, search, and 404 pages.
- Responsive header with navigation, theme switcher, and mobile layout support.
- Light, dark, and system color scheme modes.
- Telegram-inspired light and midnight blue dark color palettes.
- Built-in search page with JSON index support.
- Categories, tags, related content, and table of contents widgets.
- Article cards with cover images, metadata, categories, descriptions, and tags.
- Recommended article cards on the 404 page.
- Hugo demo site with sample posts, images, shortcodes, math examples, and about page.
- IBM Plex Sans font integration.
- SCSS and TypeScript asset pipeline for theme styles and interactive behavior.
- Social menu support with GitHub icon.
- Theme metadata, license, README, and demo configuration.
